# Y-Tian-Plugin（阴天插件）

# ※使用本插件前一定要正确配置FFmpeg※

## 介绍
Y-Tian阴天插件，一个Yunzai-Bot V3小功能集成的插件包,集娱乐，功能性于一体，让你的机器人变得更为全面多功能。（进插件群解锁更多玩法！！！）

## 安装教程

在Yunzai-Bot根目录下，运行cmd，输入以下指令
```
git clone https://gitee.com/wan13877501248/y-tian-plugin.git ./plugins/y-tian-plugin/

```
## 功能一览

| 命令| 说明|
|:--------|------------|
| #阴天更新 | 更新阴天插件 |
| #阴天帮助 | 获取相关命令|
